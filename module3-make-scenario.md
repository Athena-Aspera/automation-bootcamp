# Module 3 - Make.com Webhook to GitHub Issue

## What this scenario does
When Postman sends a POST request to the webhook URL,
Make.com receives the data and automatically creates
a new Issue in the GitHub repo.

## Tools used
- Make.com — Webhooks + GitHub modules
- Postman — to trigger the webhook
- GitHub — receives the Issue
